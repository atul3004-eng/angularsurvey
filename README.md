# survey-application

Runnable full-stack survey app (Spring Boot backend + Angular frontend).

## Quick start (Windows)

From `C:\projects\survey-application`:

```bat
run-survey-app.cmd
```

This starts:
- Backend: `http://localhost:8080`
- Frontend: `http://localhost:4200`

## Run separately

Backend only:

```bat
run-backend.cmd
```

Frontend only:

```bat
run-frontend.cmd
```

## Prerequisites

- Java 17 installed at `C:\Program Files\Java\jdk-17` (or update `run-backend.cmd` path)
- Node.js + npm
- Internet access for first dependency download
